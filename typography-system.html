05 — Hierarchy Rules
Size Jumps
Consecutive hierarchy levels must differ by at least 1 scale step. Never use --text-md directly above --text-base without a weight or family change to compensate.
Max 3 Sizes Per View
Any single screen should use no more than 3 distinct size tokens. Add hierarchy via weight, color, or tracking — not more sizes.
Negative Tracking Rule
Apply negative letter-spacing only at --text-xl and above. Below that, tracking goes neutral or positive.
Serif = Display Only
--font-display is for page headings, quotes, and hero text only. Never use it for UI labels, body, or captions — that's --font-body territory.
Color Contrast Ratio
Body text must hit WCAG AA (4.5:1). Large text 18px+ must hit 3:1 minimum. --text-quaternary is below AA — use only for truly decorative text.
Italic = Semantic
Italic is reserved for quotes, titles of works, and meaningful emphasis. Not for stylistic decoration. DM Serif italic is a design accent — use sparingly.
Uppercase Threshold
Uppercase text only at --text-xs or smaller. Uppercase body text destroys readability. Pair with --tracking-wider or --tracking-widest.
Weight Compression
On colored/active card backgrounds (like sky-300), reduce text weight by one step. --weight-bold → --weight-semibold on light surfaces.
Overline Before Heading
Overline → Heading is the canonical pattern. Overline alone has no meaning. Heading alone is valid. Never put overline after the heading it labels.
Line Height + Size
Large text needs tight leading. Small text needs open leading. --text-4xl → --leading-none. --text-sm → --leading-normal minimum.
